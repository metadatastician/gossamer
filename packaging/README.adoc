= Packaging
Platform-specific packaging recipes for distributing gossamer to end users.
Subdirectories cover `debian/` (APT), `flatpak/` (Flatpak/Flathub), `macos/` (notarised DMG), `rpm/` (RPM spec), and `windows/` (WiX installer).
