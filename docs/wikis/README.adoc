This directory is the *source of truth* for the forge-hosted wiki (GitHub Wiki).
Edit the pages here, never in the wiki UI — the UI copy is a published mirror.

== Pages (by audience)

[cols="1,3",options="header"]
|===
| Page | For

| `Home.md`        | Landing / signpost — routes each reader to their track.
| `User.md`        | App developers *using* Gossamer to build a desktop app.
| `Developer.md`   | Contributors hacking on the FFI, ABI proofs, or Ephapax core.
| `Maintainer.md`  | Release, governance, CI, and machine-readable-state owners.
| `Lay-Public.md`  | Non-technical readers (press, funders, the curious).
| `Glossary.md`    | Definitions of project-specific terminology.
|===

== Publishing

Pages are published to the forge wiki with the Justfile recipe:

[source,console]
----
just wiki-sync dry   # preview what would change
just wiki-sync       # commit + push docs/wikis/*.md to the wiki remote
----

Override the target with `GOSSAMER_WIKI_REMOTE`. The `.a2ml` manifest in this
directory is repo-only and is not published to the wiki.
